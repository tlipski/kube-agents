Every skill is a `SKILL.md` bundle in [`agents/platform/skills/`](https://github.com/gke-labs/kube-agents/tree/main/agents/platform/skills). The Platform Agent loads them on demand based on the `description:` field in each skill's frontmatter — clicking through to source below gives you the full procedure the agent follows.

For how skills are structured and invoked, see [Concepts → Skills](/kube-agents/concepts/skills/).

## Adding a new skill

See [Concepts → Skills](/kube-agents/concepts/skills/#adding-a-new-skill) for the process. New skills land under `agents/platform/skills/<name>/SKILL.md`; if they should also run on schedule, add an entry to `agents/platform/cron/jobs.json`.
